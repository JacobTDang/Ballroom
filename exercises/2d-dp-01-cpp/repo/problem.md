# Unique Paths

There is a robot on an `m x n` grid. The robot is initially located
at the top-left corner (`grid[0][0]`) and wants to reach the
bottom-right corner (`grid[m-1][n-1]`). The robot can only move
either down or right at any point in time.

Return the number of possible unique paths the robot can take to
reach the bottom-right corner.

## Example

```
Input: m = 3, n = 7
Output: 28
```

## Constraints

- `1 <= m, n <= 100`
- The answer is guaranteed to fit in a 32-bit integer.
