# Longest Palindromic Substring

Given a string `s`, return the longest palindromic substring of `s`.
If there are multiple substrings of the same maximum length, return
the first one found when scanning the string from left to right.

## Example

```
Input: s = "babad"
Output: "bab"
Explanation: "aba" is also a valid answer, but "bab" is the first
maximum-length palindrome found scanning left to right.
```

## Constraints

- `1 <= s.length <= 1000`
- `s` consists of only digits and English letters
